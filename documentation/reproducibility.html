Reproducibility Guide

This document provides detailed instructions on how to reproduce the data analysis and visualizations in the Job and Task Complexity Atlas. By following these steps, you can recreate our results, extend the analysis to different years or countries, and verify the correctness of our methodology.

Prerequisites

To reproduce this analysis, you'll need:

  1. Python 3.8+ with the following packages:
    • pandas
    • numpy
    • plotly
    • openpyxl
    • requests
    • matplotlib
    • seaborn
  2. Data Sources:
    • BLS Occupational Employment and Wage Statistics (OEWS) data
    • O*NET Database (tasks, skills, and abilities)

Installation

1. Clone the Repository

git clone https://github.com/shahzorkhan123/BLSVisualization2.git
cd BLSVisualization2

2. Install Dependencies

pip install pandas numpy plotly openpyxl requests matplotlib seaborn

Data Collection

BLS Data

The Bureau of Labor Statistics (BLS) provides comprehensive occupational employment and wage data through their OEWS program. Our analysis uses data from multiple years to track employment trends and wage distributions across occupations.

O*NET Data

The O*NET database provides detailed information about job tasks, skills, and abilities. This data is crucial for calculating job complexity metrics based on task requirements and skill demands.

Analysis Pipeline

Step 1: Data Preprocessing

Run the data preprocessing scripts to clean and standardize the BLS and O*NET data:

python scripts/preprocess_bls_data.py
python scripts/preprocess_onet_data.py

Step 2: Complexity Calculation

Calculate job complexity metrics using our custom algorithms:

python scripts/calculate_complexity.py

Step 3: Generate Visualizations

Create the interactive visualizations:

python scripts/generate_treemap.py
python scripts/generate_job_space.py

Key Metrics

Complexity Index

Our complexity index is calculated based on:

  • Cognitive skill requirements
  • Task diversity
  • Educational prerequisites
  • Experience requirements

Employment Dynamics

We track several employment metrics:

  • Employment levels
  • Wage distributions
  • Growth rates
  • Geographic concentration

Validation

To validate your results, compare your generated outputs with the reference files in the validation/ directory. Key validation points include:

  • Complexity rankings match reference rankings
  • Employment totals align with BLS official statistics
  • Visualization data matches expected patterns
Note: If you encounter any discrepancies or issues during reproduction, please check the troubleshooting.md file or open an issue on the GitHub repository.
