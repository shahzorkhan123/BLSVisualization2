Methodology

Job Complexity Index

The Job Complexity Index measures the complexity of occupations based on three key factors:

  1. Wage Range: The difference between the 90th and 10th percentile wages for each occupation, indicating the range of skill levels and specialization within the occupation.
  2. Skill Level: The average skill level required for the occupation, derived from O*NET data on the skills needed to perform the job.
  3. Task Importance: The average importance of tasks performed in the occupation, also derived from O*NET data.

The formula for the Job Complexity Index is:

Job Complexity Index = (Wage Range √ó Avg Skill Level √ó Avg Task Importance) / Median(Wage Range √ó Avg Skill Level √ó Avg Task Importance)

This index is normalized so that the median occupation has a complexity index of 1.0.

Job Price Index

The Job Price Index measures the relative price (wage) of occupations:

Job Price Index = Annual Mean Wage / Median(Annual Mean Wage)

This index is also normalized so that the median occupation has a price index of 1.0.

Task Complexity Index

The Task Complexity Index focuses specifically on the complexity of tasks performed in each occupation:

Task Complexity Index = (Avg Skill Level √ó Avg Task Importance) / Median(Avg Skill Level √ó Avg Task Importance)

This index isolates the task and skill components of complexity, separate from wage considerations.

Job Space Network

The Job Space Network visualizes the relationships between different occupations based on skill similarity:

  1. Node Creation: Each occupation is represented as a node, with size based on employment and color based on complexity.
  2. Edge Creation: Occupations are connected by edges when they share similar skill requirements, based on cosine similarity of skill vectors from O*NET data.
  3. Network Layout: A force-directed layout algorithm positions occupations with similar skills closer together, revealing clusters of related jobs.

This approach is analogous to the Product Space methodology in the Harvard Atlas of Economic Complexity, which connects products based on co-export patterns.

Data Processing

The data processing workflow includes:

  1. Extraction of BLS national and state-level occupational employment and wage data
  2. Integration with O*NET task and skill data
  3. Calculation of complexity and price indexes
  4. Aggregation at various geographic and occupational levels
  5. Generation of interactive visualizations

About the Project

The Job and Task Complexity Atlas is a research project that applies economic complexity principles to labor markets. Inspired by the Harvard Atlas of Economic Complexity, this project aims to understand the structure of labor markets through the lens of job and task complexity.

The project builds on existing research in labor economics, particularly the "tasks framework" that views jobs as bundles of tasks requiring different skills and abilities. By measuring the complexity of these task bundles and their relationship to wages and employment, we can gain insights into labor market dynamics, skill premiums, and potential paths for workforce development.

This interactive atlas allows researchers, policymakers, and the public to explore these relationships through visualizations that highlight patterns in occupational structure, geographic variations, and the relationship between complexity and wages.

Research Applications

  • Understanding wage inequality and skill premiums
  • Identifying potential career pathways and skill development opportunities
  • Analyzing regional labor market specialization and diversity
  • Forecasting potential impacts of technological change and automation
